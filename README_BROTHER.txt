[ 동생을 위한 Pakuri AI 설치 가이드 ]

안녕! 형(오빠)이 만든 "AI 비디오 누끼 툴" 설치법이야.
복잡해 보이지만 딱 3단계만 따라하면 돼.

==================================================

1. [준비물] Docker 깔기
   - 이 프로그램은 AI 뇌(서버)를 돌려야 해서 'Docker Desktop'이라는 게 필요해.
   - 구글에 "Docker Desktop" 검색해서 설치하고 실행해줘. (회원가입 안 해도 됨)
   - 실행되고 나서 아래 단계를 진행해.

2. [설치] 플러그인 넣기
   - 이 폴더에 있는 `install_plugin.bat` 파일을 더블클릭 해.
   - 검은 창이 뜨고 뭐라뭐라 하면서 "SUCCESS"라고 뜨면 성공이야.
   - (참고: 이게 프리미어 프로한테 "내가 만든 툴 쓸 수 있게 해줘"라고 허락받는 과정이야)

3. [실행] AI 서버 켜기
   - `start_pakuri_ai.bat` 파일을 더블클릭 해.
   - 처음에는 막 영어가 많이 뜨면서 다운로드 받을 거야 (시간 좀 걸림).
   - 나중에 "Uvicorn running on..." 뭐 이런 말이 뜨고 멈춰 있으면 켜진 거야.
   - **주의: 프리미어 프로 쓸 때는 이 검은 창을 절대 끄면 안 돼!**

==================================================

[사용 방법]
1. 프리미어 프로를 켠다.
2. 상단 메뉴에서 [창(Window)] -> [확장(Extensions)] -> [Smart Selector] 클릭.
3. 빨간색 경고창 뜬 패널이 나오면 성공!
4. 타임라인 영상을 클릭하고 버튼을 누르면 AI가 누끼를 따줄 거야.

문제 생기면 연락해!
- Pakuri 개발팀 (형/오빠)
